## Komentorivitoiminnot
### Testaus
Testit suoritetaan komennolla 

`mvn test`

Kattavuusraportti luodaan komennolla 

`mvn jacoco:report`

### Checkstyle
Checkstyle tarkastukset suoritetaan komennolla 

`mvn jxr:jxr checkstyle:checkstyle`
